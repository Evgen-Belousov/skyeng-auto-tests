# Автотест для Skyeng

## Описание
Автотест проверяет, что ссылка из описания личного события открывается **в новой вкладке** браузера.

## Обнаруженный дефект
Ссылка открывается в той же вкладке. Баг-репорт создан в Yozhka.

## Как запустить

### 1. Создать виртуальное окружение
```bash
python3 -m venv venv
source venv/bin/activate
### 2. Установить зависимости
bash
pip install selenium webdriver-manager
### 3. Запустить тест
bash
python3 test_link_new_tab.py
